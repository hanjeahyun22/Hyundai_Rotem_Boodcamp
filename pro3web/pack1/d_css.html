** CSS(Cascading Style Sheets) - 스타일 시트 **

    HTML 문서에 다양한 레이아웃과 꾸미기를 위한 표현적인 언어 CSS 사용
    서식 : 선택자{속성 : 값; 속성 : 값; ...}
    선택자 종류는
    1) 요소명   : body tag 내의 특정 tag를 직접 지적해서 선택
    2) .        : class 선택자 - 클래스 속성값으로 보유하고 있는 요소를 선택
    3) #        : id 선택자로 단수 선택 가능

    스타일은 하위 요소에 상속된다.
    
    CSS 적용 방법 : inline, internal, external

***CSS 적용하기 ***

-- 인라인 방식 --

인라인 방식 (해당 요소 직접 스타일 지정)

-- 인터널 방식 --

인터널 방식(head 부분에 미리 h4의 스타일 지정)

하지만, internal 방식보단, inline 방식이 우선순위!

h5 서식- 인터널 방식으로 여러 서식의 스타일 미리 지정 가능

-- class 방식 --

클래스는 동일 이름 "c1" 이 적용될 수 있음. -> class명은 중복 가능

같은 인터널 방식이더라도, 나중에 지정한 스타일이 적용됨.

-- id 방식 --

id명은 중복 불가. head에서 지정한 id에 따른 스타일 지정.

-- External 방식 : 외부 css 파일에서 서식 내용 지정--

(myid2 아이디 사용) id명은 중복 불가. external 방식 : 별도 css 파일로 적용.
